Brief

The brief for this project is to make a website using Pycharm for a topic of our choice. I choose to do this project about Wellington Queer Students Association or WQSA for short. This website will use Flask (a Python webserver), and Bootstrap 5 to provide styling in the html or the front side of the website. Then making sure the website was functional was important for me so there was a database connected to the back end.
For the database, I used SQLite on Flask to build the back end of the website. The database has multiple tables for members, events and news the members create as the last table is for people to supply their email addresses if they want to get involved.
The member table is also used for logging into the website as it holds the member's password. Wellington Queer Students Association was started up this year which is why I found the need for there to be a website for it. They wanted a place where anyone could access it and find all the information they needed about WQSA. Since it was started up this year it was at the most crucial part of putting its name out there and advertising itself. Then the final stage of this internal is making the site live.

We have previously built the foundations of a website and to develop the complexity we have to be able to manage running a database and connect it with the website. Alongside other heads of pride groups in wellington we have started an orgnaisation called Wellington queer students association to help connect and share ideas with queer youth in the greater Wellington region. Since I am head of social media to advertise and help organise I have decided to create a website for WQSA.

Plan of database

Photo of the plan of the database

Summary of code in all tables

Member table

            create table member(
                    member_id integer primary key autoincrement not null,
                    name text not null,
                    email text not null unique,
                    password text not null,
                    authorisation integer not null
                );

News

            create table news(
                    news_id integer primary key autoincrement not null,
                    title text not null unique,
                    subtitle text not null unique,
                    content text not null unique,
                    newsdate date not null,
                    member_id integer not null,
                    foreign key(member_id) references member(member_id)
                );

Comments

            create table comments(
                    comments_id integer primary key autoincrement not null,
                    comments_content text not null unique,
                    comments_date date not null,
                    member_id integer not null,
                    news_id integer not null,
                    foreign key(member_id) references member(member_id),
                    foreign key(news_id) references news(news_id)
                    );

Events

            
                create table events(
                    events_id integer primary key autoincrement not null,
                    events_title text not null unique,
                    events_content text not null unique,
                    events_date date not null,
                    member_id integer not null,
                    foreign key(member_id) references member(member_id)
                );

Email

            
                create table email(
                    email_address text not null unique,
                    email_date date not null
                );
